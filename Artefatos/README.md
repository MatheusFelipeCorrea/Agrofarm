# Artefatos do Projeto

Este diretório mantém os artefatos produzidos ao longo do projeto Agrofarm.

## Estrutura

| Pasta | Descrição |
|-------|-----------|
| **Slides/** | Apresentações do projeto |
| **Atas-Reuniao/** | Atas de reunião com o cliente |
| **Diagramas-Arquitetura/** | Diagramas de arquitetura do sistema (C4 Model Level 2) |
| **Diagramas-Casos-Uso/** | Diagramas UML de casos de uso |
| **Diagramas-Classe/** | Diagramas UML de classes (versões no README da pasta) |
| **Diagramas-Entidade-Relacionamento/** | Modelo entidade-relacionamento (MER) — versões no README da pasta |
| **Diagramas-Pacotes/** | Diagramas UML de pacotes — Frontend e Backend (cada pasta com seu README) |
| **Protótipos/** | Protótipos de tela do sistema |
