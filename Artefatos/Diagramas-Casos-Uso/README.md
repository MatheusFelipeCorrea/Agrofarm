## Escopo da versão 2.0

**Atores:** Administrador (ADMIN), Funcionário (FUNCIONARIO), Sistema (Cron Jobs).

**Principais grupos de casos de uso:**

- Autenticação e conta (login, recuperação e redefinição de senha)
- Gestão operacional — ADMIN (fazendas, culturas, colheitas, estoque, lucros, gastos, insumos, lembretes, usuários)
- Financeiro e mercado — ADMIN (cotações, simulação de exportação, histórico de simulações)
- Inteligência — ADMIN (Chat IA, insights, notícias agrícolas)
- Operação do funcionário (registro de insumos)
- Notificações in-app
- Processos automáticos (WhatsApp, cotações periódicas, arquivamento de talhões)
